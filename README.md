This project allows users to draw in the air using hand gestures
captured through a webcam.

## Features
- Draw using index finger
- Select colors using two fingers
- Eraser option
- Clear canvas
- Real-time hand tracking

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## How to Run
1. Install dependencies
   pip install opencv-python mediapipe numpy

2. Run the program
   python air_canvas.py
